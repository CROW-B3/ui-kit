# CROW-B3 UI Kit

A collection of modern React components for building stunning interfaces.

## Installation

```bash
npm install @b3-crow/ui-kit
# or
bun add @b3-crow/ui-kit
```

## Usage

Import the styles in your application:

```tsx
import '@b3-crow/ui-kit/styles.css';
```

Then import and use components:

```tsx
import { Globe } from '@b3-crow/ui-kit';

export default function App() {
  return <Globe />;
}
```
